# Speechtotext

Python package for speech to text

## [Documentation](https://jarneamerlinck.github.io/speechtotext/)

## Install package
### Python

```bash
pip install git+https://github.com/jarneamerlinck/speechtotext
```

### Conda 

```yaml
name: sample_env
channels:
dependencies:
  - python<3.11,>=3.9.2
  - pip
  - pip:
    - git+https://github.com/jarneamerlinck/speechtotext
```
