# Speechtotext

Python package to benchmark speech2text models. 

## Documentation

The documentation can be found [here](https://jarneamerlinck.github.io/speechtotext/)
## Install package
### Python

```bash
pip install speechtotext-python
```

### Conda 

```yaml
name: sample_env
channels:
dependencies:
  - python<3.11,>=3.9.2
  - pip
  - pip:
    - speechtotext-python
```
